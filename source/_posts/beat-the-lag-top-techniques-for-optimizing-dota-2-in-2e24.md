1. Press the**Windows logo key + X** together and click the**Windows PowerShell** .

**`Get-AppxPackage *SeaOfThieves* | ForEach{$_.InstallLocation}`**

4. Open the Trend Micro main console by right-clicking on the Trend Micro icon in your taskbar, and then select **Open the Main Console.**
5. Open the**Settings** menu.
6. Select **Exception Lists** , and then **Program/folders.**
7. Select the**+** button, and then **Browse.**
8. Paste Step 3 file addree into the File name box.
9. Select the **Open** button. This will add the path to your Exception List.
10. Select **OK** to save your changes and close the Exception List window.
