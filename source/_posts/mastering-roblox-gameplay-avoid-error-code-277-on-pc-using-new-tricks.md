## 2\. Disable Windows 10 Game Mode

 Game mode is a feature in Windows 10 that focuses system resources on games when enabled. It was supposed to give you a performance boost. But there are some circumstances that you would experience performance loss. To get rid of it, you‚Äôre supposed to disable Windows 10 Game Mode:

 1) On your keyboard, press the **Windows logo key**  and **I**  to open Settings.

 2) Click **Gaming** .

![disable windows 10 gaming features disable game mode](https://images.drivereasy.com/wp-content/uploads/2021/01/gaming-in-the-settings.png)

 3) From the left sidebar, select **Game Mode** . Then toggle off **Game Mode** .

![turn off game mode windows 10](https://images.drivereasy.com/wp-content/uploads/2020/11/turn-off-game-mode.jpg)

 After applying the changes, play The Division 2 to check if your issues persist. If disabling Windows 10 Game Mode didn‚Äôt do the trick, proceed to the next fix.

## 6\. Optimize in-game settings
